# Hextree POCs

This directory contains proof-of-concept (POC) projects from the Hextree Android security course.

- **AIDL_client** - AIDL (Android Interface Definition Language) client implementation
- **AttackerApp** - Attacker application for security testing
- **Broadcast_receivers** - Broadcast receiver vulnerabilities
- **ContentProvider** - Content provider security POC
- **ContentProviders** - Additional content provider examples
- **FileProvider** - File provider security POC
- **Services_pocs** - Android services security POCs
- **Webviews** - WebView security vulnerabilities
